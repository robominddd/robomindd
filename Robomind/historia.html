Un poco de su proceso

La estructura de este robot está hecha de aluminio, un material ligero y resistente que proporciona durabilidad y facilita el movimiento y la manipulación del robot. Este material es ideal para la construcción de robots, ya que permite una construcción robusta sin añadir peso innecesario.

Además, el robot cuenta con una pantalla digital, que podría servir para mostrar información relevante, como datos de funcionamiento, interacciones con el usuario o imágenes de su entorno. Dependiendo de la tecnología, esta pantalla podría ser táctil para permitir la interacción directa o tener solo fines informativos, como mostrar mensajes o indicadores del estado del robot. En este caso la pantalla muestra una cara sonriente que le da un aspecto amigable

Este robot está programado con MyCode, que es un entorno de programación basado en bloques visuales, ideal para la creación de proyectos interactivos, como robots. MyCode facilita la programación, permitiendo a los usuarios crear instrucciones sin necesidad de escribir código complejos, lo que lo hace accesible para principiantes y educativos.

En cuanto a sus capacidades, este robot tiene instrucciones específicas para que gire y detecte obstáculos gracias a su sensor de proximidad o de ultrasonido. Estos sensores le permiten detectar objetos en su camino y tomar decisiones para evitar colisiones. La programación está diseñada para que, cuando el robot detecte un obstáculo cerca, pueda girar o cambiar de dirección para continuar su recorrido sin chocar.

Funcionalidades clave del robot:

1. Giro automático: El robot gira según las indicaciones programadas. Puede estar configurado para girar a una determinada velocidad o en un ángulo específico.

2. Detección de obstáculos: El sensor de proximidad (usualmente un sensor de ultrasonido o infrarrojo) le permite detectar objetos a su alrededor. Cuando se acerca a un obstáculo, el robot toma una acción predefinida, como girar o retroceder, para evitarlo.

3. Interacción con el entorno: La pantalla digital podría mostrar información sobre la detección de obstáculos o el estado de la programación del robot.

Con estos elementos, el robot puede moverse de manera autónoma y realizar tareas específicas, todo basado en la programación simple a través de MyCode.
